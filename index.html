Automated Daily Commits with Mathematical Constants

This repository is set up to automate daily commits with various mathematical constants, including PI, PHI, Euler's number, and the Golden ratio.

The scripts in this repository calculate and store the following values:

  • PI: a Python script that calculates and stores the value of PI
  • PHI: a Python script that calculates and stores the value of PHI
  • Euler's number: a Python script that calculates and stores the value of Euler's number, and a C program that calculates and prints the value of Euler's number
  • Golden ratio: a Python script that calculates and stores the value of the Golden ratio
  • Fibonacci sequence: a Python script that calculates and prints the Fibonacci sequence, and a SQL script that creates a table for the Fibonacci sequence

To set up this repository, simply run the following command:

bash -c "curl -s https://raw.githubusercontent.com/your-username/your-repo-name/master/setup.sh | bash"

Replace your-username and your-repo-name with your actual GitHub username and repository name.

This script will set up a Heroku app and GitHub repository to automate daily commits with the specified mathematical constants.
